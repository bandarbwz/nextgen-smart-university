# NextGen Smart University Platform (NSUP)

A modern enterprise-level web-based university management platform.

---

# Project Overview

NextGen Smart University Platform (NSUP) is a complete web-based university management system that integrates academic services, student services, campus activities, communication, and administration into one platform.

The platform is designed for universities with up to 50,000 users.

---

# Main Modules

- Authentication
- Student Portal
- Lecturer Portal
- Coordinator Portal
- Administrator Portal
- Restaurant Portal
- Learning Management System (LMS)
- Smart Course Registration
- QR Attendance
- AI Exam Monitoring
- Student Activities
- Food Court

- Calendar
