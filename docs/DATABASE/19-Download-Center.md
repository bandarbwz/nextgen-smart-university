# Download Center Database

## Purpose

This document defines the database structure for downloadable files generated by the platform.

---

# Overview

The Download Center stores references to all downloadable documents.

---

# Tables

## downloadable_files

Fields

- id
- title
- category
- file_path
- file_type
- owner_id
- generated_at
- expires_at

---

## download_history

Fields

- id
- file_id
- user_id
- downloaded_at
- ip_address

---

# Categories

- Transcript
- Attendance
- Grades
- Registration
- Student ID
- Certificate
- Excuse Letter
- Invoice
- Receipt
- Report

---

# Relationships

downloadable_files

1 → Many

download_history

---

users

1 → Many

download_history

---

# Business Rules

- Users may only download authorized files.
- Every download is logged.
- Expired files cannot be downloaded.

---

# Notes

Files are generated automatically by Academic, Finance, Reporting, and Student Activities modules.
