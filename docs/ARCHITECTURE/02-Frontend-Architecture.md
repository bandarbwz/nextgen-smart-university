# Frontend Architecture

## Overview

The frontend provides a responsive web interface for all platform users.

It is built using React, Bootstrap, and modern frontend development practices.

---

# Technology Stack

- React
- React Router
- Bootstrap
- Axios
- Socket.IO Client

---

```

├── pages/

├── assets/

```
