# Notification System

## Overview

The Notification System delivers real-time updates to all users across the platform.

Notifications help users stay informed about academic activities, exams, food orders, financial updates, university events, and system announcements.

---

# Objectives

The Notification System provides:

- Real-time notifications
- Reminder notifications
- Emergency announcements
- Role-based notifications

---

# Notification Types

Academic

- Course Registration
- Course Drop
- Timetable Changes

Attendance

- Attendance Recorded
- Absence Warning
- Excuse Approved

LMS

- New Assignment
- Assignment Deadline
- Quiz Published
- Grade Released

Food Court

- Order Accepted
- Order Ready
- Payment Successful

Student Activities

- Event Registration
- Event Reminder
- Certificate Available
- Reward Points Added

Finance

- Tuition Reminder
- Payment Successful
- Refund Approved

System

- Maintenance Notice
- Security Alert
- Password Changed

---

# User Controls

- Search Notifications

# Notification Priority

Levels

- Low
- Normal
- High
- Critical
