# Reporting System

## Overview

The Reporting System provides detailed reports and analytics for students, lecturers, coordinators, administrators, STAD staff, and restaurant owners.

Reports help users monitor performance, analyze data, and make informed decisions.

---

# Objectives

The Reporting System provides:

- Academic Reports
- Attendance Reports
- LMS Reports
- Financial Reports
- AI Exam Reports
- Food Court Reports
- Student Activity Reports
- System Reports

---

# Student Reports

Students can view:

- Academic Transcript
- GPA Report
- Attendance Report
- Assignment Report
- Quiz Report
- Payment History
- Event Participation
- Reward Points
