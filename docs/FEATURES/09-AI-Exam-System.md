# AI Exam System

## Overview

The AI Exam System provides intelligent online exam monitoring using artificial intelligence and computer vision.

It helps maintain academic integrity by monitoring student behavior during online exams while allowing lecturers to review all detected violations before making final decisions.

---

# Objectives

The AI Exam System provides:

- Identity Verification
- Live Camera Monitoring
- Eye Tracking
- Face Detection
- Multiple Face Detection
- Fullscreen Monitoring
- Browser Tab Detection
- AI Violation Reports
- Exam Recording
- Lecturer Review

---

# Student Workflow

Before Exam

Students must:

- Login
- Verify Identity
- Enable Camera
- Enable Microphone
- Allow Browser Permissions
- Enter Fullscreen Mode

System checks:

- Camera availability
- Internet connection
- Student enrollment
- Exam availability

---

# During Exam

The AI continuously monitors:

- Face Visibility
- Eye Direction
- Head Position
- Camera Status
- Fullscreen Mode
- Browser Tab Switching
- Session Activity

---

# AI Violations

The system detects:

- Face Not Visible
- Multiple Faces
- Looking Away Frequently
- Camera Disabled
- Fullscreen Exit
- Browser Tab Change
- Suspicious Behavior

Each violation includes:

- Time
- Screenshot
- Confidence Score
- AI Recommendation

---

# Lecturer Review

Lecturers can:

- View Violation Timeline
- Review Screenshots
- Watch Recordings
- Ignore False Positives
- Confirm Violations
- Approve Final Result

---

# Student Notifications

Students receive warnings when:

- Camera is disabled.
- Face is not detected.
- Fullscreen is exited.
- Multiple violations occur.

---

# Security

The system requires:

- Camera Permission
- Microphone Permission
- Secure Browser Session
- JWT Authentication
- Encrypted Communication
