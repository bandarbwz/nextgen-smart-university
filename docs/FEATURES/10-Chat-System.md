# Chat System

## Overview

The Chat System provides secure real-time communication between students, lecturers, coordinators, administrators, restaurant owners, and STAD staff.

The platform replaces external messaging applications for academic communication.

---

# Objectives

The Chat System provides:

- Course Chat Rooms
- Private Chats
- Group Chats
- File Sharing
- Image Sharing
- Video Sharing
- Voice Messages
- Message Reactions
- Read Receipts

---

# Course Chat

Each course automatically has its own chat room.

Students are:

- Added automatically after course registration.
- Removed automatically after dropping the course.

---

# Private Chat

Users can start private conversations with:

- Lecturers
- Coordinators
- Administrators
- STAD Staff
- Restaurant Owners (Food Orders)

---

# Messaging Features

Users can:

- Send Text Messages
- Edit Messages
- Delete Messages
- Reply to Messages
- Forward Messages
- React to Messages

---

# Media Sharing

Supported media:

- Images
- Videos
- Documents
- PDF Files
- Office Files
- Audio Files

---

# Lecturer Features

Lecturers can:

- Pin Important Messages
- Share Lecture Files
- Make Announcements
- Moderate Course Chats

---

# Search

Users can search by:

- Message
- Sender
- File Name
- Date

---

# Notifications

Notifications include:

- New Message
- Mention
- Reply
- Shared File
- Lecturer Announcement

---

# Security

The system provides:

- Secure Authentication
- File Validation
- File Size Limits
- Malware Scanning
- Permission Validation

# Success Criteria

The Chat System is complete when all academic communication can be performed securely within the university platform without relying on third-party messaging applications.
