# Accessibility

## Overview

The platform follows accessibility best practices to ensure usability for all users.

---

# Accessibility Goals

- Keyboard Navigation
- Screen Reader Support
- High Contrast
- Responsive Text
- Clear Focus Indicators

---

# Keyboard Support

Users can navigate using:

- Tab
- Shift + Tab
- Enter
- Escape
- Arrow Keys

---

# Color Contrast

All text meets WCAG contrast recommendations.

---

# Icons

Every icon includes an accessible label.

---

# Images

All images include descriptive alternative text.

---

# Forms

Every input includes:

- Label
- Placeholder
- Validation Message

---

# Error Messages

Errors clearly explain:

- What happened
- How to fix it

---

# Accessibility Standards

Target Standard

WCAG 2.1 AA

---

# Success Criteria

Every page is accessible to users with different abilities and supports keyboard-only navigation.
